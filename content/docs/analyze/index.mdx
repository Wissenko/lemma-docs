## What Good Analysis Does

- separates repeated patterns from one-off anecdotes
- keeps exact quotes connected to transcripts
- preserves participant context instead of flattening everything into averages
- turns findings into a decision, recommendation, or next question

## Example Study

The audited example, "How knowledge workers adopt AI tools", contains completed sessions, participant summaries, transcripts, a study chat surface, Doc output, and Slides output. Use it as a model for what a complete Lemma workflow should produce.
