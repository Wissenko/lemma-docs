Topics are subscription categories for participants.

The app describes them as categories participants can opt in or out of. Distribution can use topics to gate sends to relevant audiences only.

<ProductScreenshot
  src="/screenshots/product-audit/07-contact-topics.png"
  alt="The Lemma Topics settings screen."
  caption="Topics are useful when different contacts should receive different kinds of studies."
/>

## Create a topic

The topic form includes:

- key
- label
- description

<ProductScreenshot
  src="/screenshots/product-audit/12-new-topic-modal.png"
  alt="The Lemma New topic form."
  caption="Use a stable key and a human-readable label for each topic."
/>

## Examples

- `ux-research`
- `pricing`
- `beta`
- `customer-stories`
