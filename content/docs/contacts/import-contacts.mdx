## Before You Import

At minimum, include an email column.

Recommended columns:

- email
- first name
- last name
- phone
- role
- company
- segment or audience label

## Supported Files

The import screen accepts:

- CSV
- TSV
- XLSX
- XLS
- XLSB
- ODS

The observed limit was up to **5000 rows per sheet**.

<HelpCallout type="tip" title="Prepare reusable columns">
If you already know how you will target studies, add those fields before importing. They can become properties and segment filters later.
</HelpCallout>

## After Import

Use segments, topics, and properties to organize contacts for future studies.

## FAQs & Common Issues

### What column is required?

Use email as the minimum identifier unless your workspace has a different verified import rule.

### Should I clean the list before importing?

Yes. Remove duplicates, obvious test records, and columns you will not use for targeting or context.

### What should I do after import?

Create [properties](/contacts/properties) for custom fields you want to reuse.
