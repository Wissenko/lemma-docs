## Fields

The audited form included:

- email
- first name
- last name
- phone

## When to use manual entry

Use manual entry when:

- you only need to add one person
- you are testing contacts
- the participant is not part of a list yet

Use [import](/contacts/import-contacts) when you have many contacts.
