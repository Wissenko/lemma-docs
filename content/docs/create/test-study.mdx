Test the study before using it with participants. This helps you catch questions that are too broad, too shallow, or pointed in the wrong direction.

## How To Test A Study

<Steps>
<Step>

### Step 1: Finish the draft

Make sure the study context and interview questions are generated.

</Step>
<Step>

### Step 2: Open Test Interview or Review

Use the testing or review action from the study builder.

</Step>
<Step>

### Step 3: Check the first question

The first question should match the study goal and feel specific to the participant.

</Step>
<Step>

### Step 4: Check the follow-ups

Make sure follow-ups ask for useful detail instead of repeating the same idea.

</Step>
<Step>

### Step 5: Edit and test again

Update the context, questions, key terms, or follow-up instructions, then test again if needed.

</Step>
</Steps>

## What To Look For

- the opening feels clear
- the first question is not generic
- the follow-ups ask for useful detail
- key terms are recognized
- the closing message fits the study

## FAQs & Common Issues

### What if the first question feels generic?

Tighten the study goal or rewrite the question.

### What if follow-ups go in the wrong direction?

Add clearer follow-up instructions for that question.

### What if proper nouns are not recognized?

Add them as key terms.
