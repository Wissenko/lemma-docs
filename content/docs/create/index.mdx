## FAQs & Common Issues

### Do I need to write all questions first?

No. You can start with a rough goal in Smart Builder.

### Can I use an existing discussion guide?

Yes. Use **Upload Discussion Guide** from the New Study screen.

### What should I review before using the study?

Review the generated context, questions, follow-up settings, key terms, welcome message, and closing message.
