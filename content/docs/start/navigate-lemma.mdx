Lemma is organized around studies, contacts, analysis, outputs, and workspace settings.

## Main navigation

The left sidebar includes:

- **New Study**: create a new study.
- **Studies**: search, open, duplicate, pause, archive, or delete studies.
- **Contacts**: manage participants, imports, segments, topics, and properties.
- **Workspace**: manage organization settings and team members.
- **Account & Billing**: manage profile, notifications, credits, plans, and billing.
- **Support**: open support chat.

## Where to go first

<TwoColumn>
  <Panel title="I want to run a new study">
    Start with New Study, then choose text or voice and follow-up depth.
  </Panel>
  <Panel title="I want to manage participants">
    Start with Contacts, then import contacts or build segments.
  </Panel>
  <Panel title="I want to review data">
    Open a study, then use Responses, Sessions, transcripts, and Chat.
  </Panel>
  <Panel title="I want a shareable output">
    Open the study, then use Doc or Slides.
  </Panel>
</TwoColumn>

## Study-level navigation

Inside a published study, the audited example showed:

- **Responses**
- **Doc**
- **Slides**
- **View survey**

The Responses area also included:

- **Overview**
- **Sessions**
- **Chat**
