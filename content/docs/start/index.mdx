## FAQs & Common Issues

### Should I start by importing contacts?

Not always. For a first test, you can create a study and share a link manually. Use Contacts when the audience needs to be reusable or targeted.

### Should my first study be voice or text?

Use voice when nuance, stories, emotion, or hesitation matter. Use text when you need a quieter, lower-friction participant path.
