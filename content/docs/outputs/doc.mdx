## What the Doc can include

In the audited example, the Doc included:

- study title and objective
- sample size
- method
- field period
- issue date
- completed interviews
- recorded conversation time
- distinct functions
- quotes available
- exhibits
- key findings
- participant quotes
- recommendations

## When to use Doc

Use Doc when you need:

- a research memo
- an evidence-backed summary
- a shareable artifact for stakeholders
- a source for product, sales, marketing, or strategy decisions
