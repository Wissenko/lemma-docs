## Basic Workflow

<Steps>
<Step>

### Create the study

Start from a brain dump, template, or discussion guide. Lemma can draft the study context, questions, and follow-up guidance.

</Step>
<Step>

### Refine the setup

Choose text or voice, choose follow-up depth, edit the questions, and add instructions for follow-ups.

</Step>
<Step>

### Test the study

Use Test Interview or Review before using the study with real participants.

</Step>
<Step>

### Review the results

Use Responses for sessions, Transcript for source wording, Insights for synthesis, and Research for targeted questions.

</Step>
</Steps>

## FAQs & Common Issues

### Is Lemma only for voice studies?

No. A study can use text or voice.

### Where should a new user start?

Start with [How do I create my first study?](/start/quickstart).

### What should I open after participants respond?

Open [How do I review results?](/analyze).
