Workspace settings control the shared workspace, not only your personal account.

The audited screen included:

- organization name
- Save button
- Team Members section
- invite members
- member search
- member role table

## When to use this page

Use Workspace settings when:

- the organization name is wrong
- you need to invite teammates
- you need to inspect who has access
- you need to understand member roles

## Screenshot note

The workspace settings screenshot captured during audit contains private member data. Create a redacted screenshot or a clean demo workspace before publishing one here.
