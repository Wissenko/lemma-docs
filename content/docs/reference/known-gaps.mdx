This page tracks documentation gaps found during the product audit.

## Participant flow

The read-only example study had a **View survey** button, but the route returned a 404 during the audit.

Needed:

- capture a working participant landing page
- capture a text conversation flow
- capture a voice conversation flow
- verify microphone permission copy

## Export

Doc and Slides both showed an **Export** button.

Needed:

- confirm export file formats
- confirm whether export creates files, downloads, or links
- confirm whether export consumes credits

## Research Agent

The chat surface was visible, but no prompt was sent.

Needed:

- confirm whether prompts consume credits
- capture an example answer in a demo study
- document how to cite or verify Research Agent answers

## Billing

The app shows audio credits and text interviews.

Needed:

- confirm exact credit consumption rules
- confirm overage behavior
- confirm top-up behavior if any

## Workspace roles

The member table shows roles, but the full role model was not verified.

Needed:

- list all roles
- define what each role can do
- explain participant vs teammate records
