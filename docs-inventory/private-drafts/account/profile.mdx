## What You Can Change

The audited profile section showed:

- display name
- email
- password reset link

You can edit your display name.

The email address is displayed as account information.

Use **Send reset link** to start a password reset flow.

## Profile vs Workspace Settings

Use Profile for personal identity and account access.

Use [Workspace settings](/account/workspace-settings) for organization-level details and team access.

## FAQs & Common Issues

### Can I change my workspace name here?

No. Use [Workspace settings](/account/workspace-settings).

### Where do I change notification preferences?

Use [Notifications](/account/notifications).

### Where do I reset my password?

Use **Send reset link** from the profile section.
