## When To Use Manual Entry

Use manual entry when:

- you only need to add one person
- you are testing contacts
- the participant is not part of a list yet

Use [import](/contacts/import-contacts) when you have many contacts.

## Fields

The audited form included:

- email
- first name
- last name
- phone

## FAQs & Common Issues

### Should I add test contacts manually?

Yes. Manual entry is useful for small tests before you import a larger audience.

### What if I need to add many people?

Use [How do I import contacts?](/contacts/import-contacts).

### What if I need to target this person later?

Add properties or include the contact in a segment once the relevant audience rules are clear.
